Línea de Tiempo de las Clases

  • Semana 1 - Introducción al curso asincrónico de JavaScript

    Los fundamentos y principios básicos de JavaScript.

  • Semana 2 - Estructuras de Datos y Manipulación de DOM

    Dominar las estructuras de datos y aprender a interactuar con el DOM para manipular elementos en una página web.

  • Semana 3 - Programación Asincrónica y Manejo de APIz

    Trabajar con código asincrónico y consumir APIs exterbas usando JavaScript.

  • Semana 4 - Proyecto Final

    Desarrollo de un proyecto utilizando todo lo aplicado en las semanas anteriores.

Resumen del Curso

Este curso ofrece una introducción completa a la robótica, desde los conceptos básicos hasta la creación de un proyecto final con el uso de sensores, actuadores y controladores. Aprenderás acerca de todo lo relacionado a robótica, con un profesor de calidad y consejos que puedes aplicar en tus proyectos para darle vida a tu robot.
